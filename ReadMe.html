Python Project to PyPI Tutorial

  1. Clone Git Repository: 
    git clone https://github.com/your_username/your_project.git
    cd your_project
  2. Write Your Python Code: Add your Python code files to the project.
  3. Create __init__.py File: Make sure to create an empty __init__.py file in each package directory.
  4. Create Your Python Project Structure: Organize your project with packages and modules, and make sure to have a proper directory structure.
  5. Install Wheel: pip install wheel
  6. Create setup.py File: 
    from setuptools import setup, find_packages

setup(
    name='your_lib_name',
    version='0.1.0',
    packages=find_packages(),
    install_requires=[
        # List your dependencies here
    ],
)
  7. Create Source Distribution and Wheel: python setup.py sdist bdist_wheel
  8. Install Twine: pip install twine
  9. Create PyPI Account: Go to PyPI and create an account if you don't have one.
  10. Generate PyPI Token: Go to your PyPI account settings, create an API token, and copy it.
  11. Upload to PyPI Using Twine: 
    twine upload --username __token__ --password <your_token> dist/*
  12. Install Your Library: pip install your_lib_name
